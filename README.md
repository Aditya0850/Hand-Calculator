🖐️ AI Hand Gesture Calculator

A real-time computer vision based calculator that performs arithmetic operations using hand gestures.
Built using OpenCV, MediaPipe, and Python.

This project detects fingers using a webcam and interprets gestures to perform mathematical operations with optional voice feedback.

🚀 Features

✋ Real-time hand detection using MediaPipe

🔢 Finger counting logic

➕ Gesture-based operator selection

1 finger → Addition

2 fingers → Subtraction

3 fingers → Multiplication

4 fingers → Division

🎤 Voice feedback using pyttsx3

👥 Two-hand number detection

🔄 Restart and exit controls

🛠️ Tech Stack

Python

OpenCV

MediaPipe

pyttsx3 (Text-to-Speech)
